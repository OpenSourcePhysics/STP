Statistical and Thermal Physics Programs

This collection includes all programs used in the text Statistical and Thermal Physics: With Computer Applications, 2nd ed., Harvey Gould and Jan Tobochnik, Princeton University Press (2021). The Java programs use the Open Source Physics (OSP) library, created by Wolfgang Christian and supported by many other collaborators.

Program Resources and Organization
  1. On each program page the user can run the program from their browser. Included are problems from the textbook. The programs were converted from Java to JavaScript by Wolfgang Christian and Robert Hanson using the SwingJS system developed at St. Olaf College.
  2. Source code for all Java programs can be downloaded. Each program source code appears in the chapter folder where it is first used. The Java programs require the OSP library to run. For some problems such as those using Monte Carlo and Molecular Dynamics simulations of many particles, the Java programs run much faster than the JavaScript or Python versions.
  3. Source code for all Python programs can be downloaded. Each program source code appears in the chapter folder where it is first used. For novice programmers the Python code may be easier to follow and modify than the Java code.
  4. Download executable jar file (need to install Java on your computer to run jar files.) This jar file is organized in a way similar to that of this Compadre collection.
