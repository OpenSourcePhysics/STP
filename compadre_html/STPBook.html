Statistical and Thermal Physics Programs

This collection includes all programs used in the text Statistical and Thermal Physics: With Computer Applications, 2nd ed., Harvey Gould and Jan Tobochnik, Princeton University Press (2021). The Java programs use the Open Source Physics (OSP) library, created by Wolfgang Christian and supported by many other collaborators. The source files for these programs can be downloaded. They are organized according to the first chapter they are used. The OSP library must also be included.

The JavaScript version of these programs can be run from the web pages in the menu bar. They are based on the Java programs

Python versions of the programs are available for download.

Individual jar files which are created from the Java programs are found in the program web pages reached from the menu bar. These versions run much faster than the JavaScript and Python versions.
