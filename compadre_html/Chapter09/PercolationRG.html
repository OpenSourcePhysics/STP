Program PercolationRG

The key idea of a renormalization group (RG) transformation for percolation is to average over small parts of the lattice and then determine the effective value of $p$ for the new lattice. In this program we do the averaging by taking a $2 \times 2$ block of sites and replace it with a single occupied site if the occupied sites in the $2 \times 2$ block span vertically. Otherwise the blocked site is not occupied. Then the blocked spins are shown. This process is repeated using the blocked spins. The goal of the program is to allow the user to visualize the RG process.
