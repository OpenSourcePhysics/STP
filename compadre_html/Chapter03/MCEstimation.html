Program MCEstimation

When we want to compute the average of a quantity that can take on many values, it is usually impossible to average over all possible values. Instead we can sample the possible values and determine an appropriate average. This procedure is referred to as a Monte Carlo (MC) calculation.

Program MCEstimation illustrates a simple example of a MC calculation by estimating the integral $$\int_0^1 y(x) dx = \int_0^1 \sqrt{1 - x^2} dx,$$ where the possible values of $(x,y)$ are within a $1 \times 1$ box. The program randomly generates pairs $(x_i,y_i)$ and counts the percentage where $y_i \leq y(x_i)$. Because the area of a $1 \times 1$ box is unity, this percentage provides an estimate of the integral.
