Statistical and Thermal Physics Programs

This collection includes all the Java programs used in the text Statistical and Thermal Physics: With Computer Applications, 2nd ed., Harvey Gould and Jan Tobochnik, Princeton University Press (2021). The Java programs use the Open Source Physics (OSP) library created by Wolfgang Christian and supported by many other collaborators. The source code for the Java programs and OSP can be found on Github.

Program Resources and Organization

  1. Each STP Javascript program can be run in a browser.
  2. Source code for all Java and Python versions of the programs, and the latest version of this jar file can be found on ComPADRE.

We thank Wolfgang Christian for his assistance in creating this jar file. The Javascript programs on ComPADRE were converted from Java by Wolfgang Christian and Robert Hanson using the SwingJS system developed at St. Olaf College.
